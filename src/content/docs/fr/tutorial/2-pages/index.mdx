Maintenant que vous avez un site web fonctionnel en ligne, ajoutons des pages et des articles !

## Où en sommes-nous ?

Vous pouvez trouver le code à ce stade du tutoriel sur [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-2/start) ou
[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-2/start?file=src/pages/index.astro).

Vérifiez votre code par rapport à cet exemple, ou si vous venez de nous rejoindre, faites un fork sur StackBlitz pour commencer à coder directement depuis votre navigateur à partir d'ici.

## Où allons-nous ?

Dans cette unité, vous allez :
- Créer vos premières pages Astro avec la syntaxe `.astro`
- Ajouter des articles de blog avec des fichiers Markdown (`.md`)
- Styliser une page individuelle avec `<style>` 
- Appliquer des styles globaux sur l'ensemble des pages

En cours de route, vous apprendrez comment les **deux sections d'un fichier `.astro`** travaillent ensemble pour créer une page, ainsi que comment utiliser des variables et un rendu conditionnel sur vos pages.
